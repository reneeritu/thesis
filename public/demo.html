aura2 — minimal demo (matches API / Zod)

Host this page on the same origin as the API (e.g. deploy API on Render, then open /demo.html). Or set API base to your Render URL — CORS is open by default.

1 — Register (POST /auth/register)

alias: 3–30 chars, lowercase letters, numbers, _ - only. password: 8–128 chars.

2 — Login (POST /auth/login)

3 — Space (POST /spaces)

4 — Join space (POST /spaces/:id/join) — second user

Use a second account’s token above. Open space must be open, or provide invite code.

5 — Project / START (POST /projects)

6 — Trace (POST /traces)
